Abaixo são descritas classes com seus respectivos atributos e tipos de dados. 
Escolha duas classes, e para cada uma das classes escolhidas, desenvolva:
a classe de negógio em java (mapeada para o hibernate) e suas 4 telas CRUD 
(Tela de Cadastro, Tela de Alteração, Tela de Exclusão e Tela de Listagem).
Utilize como base o projeto java do netbeans ProjetoBancoABC_Hibernate.
